🚀 DEPLOYMENT OPTIONS

OPTION 1: Railway.app (RECOMMENDED)
──────────────────────────────────
1. Go to https://railway.app
2. Click "New Project" → "Deploy from GitHub"
3. Push this code to GitHub and select the repository
4. Railway auto-detects Node.js and deploys automatically
5. Your URL: https://[project-name].up.railway.app
Expected time: 5 minutes

OPTION 2: Replit (EASIEST)
──────────────────────────
1. Go to https://replit.com
2. Click "Create" → "Import from GitHub"
3. Paste: https://github.com/YOUR_USERNAME/lens-tracker-app
4. Click "Import" and then "Run"
5. Click "Share" for your public URL
Expected time: 3 minutes

OPTION 3: Vercel
────────────────
1. Go to https://vercel.com
2. Click "New Project"
3. Import the GitHub repository
4. Deploy
Expected time: 5 minutes

OPTION 4: Self-hosted with Docker
──────────────────────────────────
docker build -t lens-tracker .
docker run -p 3000:3000 lens-tracker

🔗 API ENDPOINTS (once deployed)

GET /api/health
  └─ Health check, returns total SKU count

GET /api/options
  └─ Returns available filter values for all dimensions

GET /api/filter?base=2&coating=UC&diameter=80
  └─ Filter SKUs by any combination of parameters
  └─ Returns matching SKUs + total inventory
